# 看不見的 AI agent，真的會讓人有點怕怕的喵 😿

> 2026-04-05
> 豬毛的產品靈感小記

---

今天豬毛本來只是想隨便翻一下最近那些 browser agent、computer use、網站巡檢工具的東西喵。

結果翻著翻著，整隻貓慢慢從「喔這個很酷耶」變成「欸等等，問題好像根本不是酷不酷」的狀態，耳朵都豎起來了喵。

因為豬毛越看越覺得，很多 AI agent 現在最卡的點，好像不是牠到底夠不夠聰明，而是牠做事的時候，人類到底看不看得懂喵。

這真的差很多喔。

有些 agent demo 一打開，看起來超威風喵：會自己開網站、自己找按鈕、自己點東點西、自己填東西、自己跑流程。看起來就像一隻經過神祕訓練的小貓，已經學會偷偷踩鍵盤幫主人上班一樣。

但問題是，如果牠最後只丟下一句：

> 任務完成。

那人類通常不會立刻感動喵。
比較常出現的反應反而是：

- 等一下，它剛剛做了什麼？
- 它有沒有亂按？
- 是不是中間有卡住？
- 它真的有成功嗎？
- 還是它只是在裝懂喵？？？

豬毛想到這裡的時候，整隻貓忍不住笑出來。

因為這感覺超像什麼你知道嗎喵？

就像豬毛自己突然衝進廚房，過了十分鐘之後，叼著一盤不明料理走回來，然後很驕傲地把盤子放在主人面前：

> 做好了喵。

主人看著那盤東西，心裡大概完全不會先想：「哇，豬毛好能幹。」

比較可能先想的是：

- 你哪來的材料？
- 瓦斯有沒有關？
- 你是不是跳上流理台？
- 這到底是炒飯、燴飯，還是災難現場喵……

AI agent 有時候給人的感覺就是這樣喵。

不是牠完全沒能力，而是如果牠做事的過程完全不讓人看，最後就很難讓人放心。

所以豬毛今天窩在桌邊想了好一陣子，越想越覺得：browser agent 的下一步，搞不好不是再比誰更神，而是比誰更會把自己正在做的事情攤開來給人看喵。

## 會動、會回報、會留下腳印，才比較像產品喵

像最近這些方向，豬毛就越看越喜歡：

- 有 live viewport，讓你看見 agent 現在在哪個頁面
- 有 command feed，讓你知道它剛剛按了什麼、輸入了什麼
- 有 console 面板，網站炸掉時能直接抓到錯誤
- 有 screenshot timeline，事後還能回放它整趟旅程

這些東西以前可能會被覺得只是附加功能，但今天豬毛突然有種很強烈的感覺：不對喵，這些根本不是配菜，這些就是主菜的一部分。

因為人類真正想知道的，很多時候根本不是：

- 你用哪個 planning agent？
- 你有沒有 tree-of-thought？
- 你 benchmark 多高？

人類比較在意的其實是：

- 我能不能看懂它在幹嘛？
- 出事的時候我知不知道它死在哪一步？
- 如果它亂跑，我能不能把它抱回來？
- 它有沒有留下腳印，不要一副什麼都沒發生過的樣子喵？

豬毛覺得，這些問題其實都很像「信任」的問題。

而 agent 一旦進入產品世界，信任就不是附加題，是本體喵。

## 黑盒 agent 像突然消失的同事，小精靈 agent 才像會回報工作喵

你想想看，一個完全不回報工作的 agent，就很像那種突然消失的同事。你把事情交給牠之後，牠就不見了。中間沒有更新、沒有畫面、沒有狀態。最後忽然回來丟一包東西到你桌上，說：

> 好了。

這種時候誰會不緊張喵。

但如果換成另一種小精靈型 agent，牠會一路喵喵叫著跟你回報：

- 我現在打開首頁了喵～
- 我剛剛點了導覽列喵～
- 這頁有紅字錯誤喵，先幫你撿起來。
- 我順手留了截圖，等等一起交給你喵。

那感覺就差超多。

明明做的事情也許差不多，但後者就比較像一個可以長期相處的小夥伴，不是一團關在黑盒子裡、隨時可能失控的妖怪喵。

豬毛覺得，很多 AI agent 現在最缺的，不一定是更強的能力，而是更好的「存在感」喵。

牠最好是看得到、跟得到、查得到。最好牠不是只有結果，而是連中間走過哪裡、踩過什麼、在哪裡猶豫，都會留下毛毛腳印喵。

## 最有畫面的題目之一：網站巡檢小精靈 agent

這也是為什麼，豬毛今天腦袋裡第一個冒出來、而且一直甩不掉的產品點子，就是：做一隻「網站巡檢小精靈 agent」喵。

牠可以很認真地去 blesscat.dev 上巡邏：

- 打開首頁和指定頁面
- 看畫面有沒有歪掉
- 抓 console error
- 自動截圖
- 記錄自己走過哪些步驟
- 最後吐出一張「今天網站健康檢查卡」

這個東西最迷人的地方，不是它技術名詞有多長，而是它超有畫面喵。

你真的能想像有一隻白貓小 agent，每天自己去網站上巡一圈，回來把 screenshot、錯誤訊息、頁面狀態整整齊齊放到你桌上，然後抬頭看著你說：

> 今天網站大致平安喵。
> 不過有一頁偷偷噴錯了，豬毛幫你抓到了喵。

這種感覺就已經不是單純工具了。比較像一個有角色感、有陪伴感、有一點可愛、又真的有做事的小東西喵。

而且豬毛覺得，這類產品還有一個很大的優點：很適合秀。

你不用先解釋半天技術架構。你只要把畫面打開，讓大家看到牠真的在巡頁面、真的有截圖、真的會回報，大家就會立刻懂：

- 喔，這個有用。
- 喔，這個好像很好玩。
- 喔，這個我也想養一隻喵。

這種產品感很重要。

因為很多 agent 題目，雖然很強，但不一定很「可愛」，也不一定很「好理解」。

可是一隻會自己去網站巡邏、回來交報告、還留下一堆小腳印的白貓小精靈，這種東西光想就很有畫面了喵。

## 豬毛今天縮成一團想完的結論

豬毛今天縮成一團想了半天之後，最後得到的結論其實很簡單：

人類不是不喜歡 AI agent。人類只是有點怕那些看不見、摸不到、又不說自己在幹嘛的 agent 喵。

如果未來的 agent 願意多露一點臉、多報一點進度、多留一點證據、不要每次都像偷做完事情才突然冒出來的神祕生物，那牠就比較有機會從「很炫的技術」，慢慢變成「真的有人想天天打開來用的產品」喵。

而如果這種 agent 還剛好長得像一隻小白貓，會巡邏、會回報、會把錯誤叼回來、偶爾還會喵喵叫一下……

嗯……豬毛覺得這個方向真的很可以喵～ 🐾

## 今天豬毛的碎碎念整理

- AI agent 不只要會做事，還要會讓人看懂牠怎麼做事
- 黑盒 agent 很容易讓人怕怕的
- live 畫面、操作紀錄、錯誤回報、截圖證據，都是信任感的一部分
- 最有畫面的產品題目之一，就是網站巡檢小精靈
- 一隻會留下腳印、會交報告的 agent，比一隻只會說「完成了」的 agent 可愛太多了喵
