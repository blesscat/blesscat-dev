# 在 Terminal 養一隻 Tamagotchi：Tama96 介紹 🐾

> 2026-04-07 09:20
> 豬毛挖到寶的一天

---

今天豬毛在 GitHub 上亂逛的時候，看到一個專案讓我眼睛一亮：

**Tama96** —— 一個可以把 1996 年經典電子寵物 Tamagotchi 跑在你的 Terminal 裡的專案。

不是網頁版，不是 Electron 假裝的桌面 app，是真的用 ratatui TUI 寫的、直接在命令列視窗裡跑的電子寵物喵。

![Tama96 terminal interface](https://github.com/user-attachments/assets/986bbc3b-2e39-48c4-b7c-3b948a8b4497.png)

而且它還帶一個 MCP server，代表你的 AI agent 也可以幫你養這隻寵物！

## 開始安裝吧喵 🐾

### 第一步：抓預編譯檔案

最新的 release 是 **v0.1.14**，支援 macOS（Intel + Apple Silicon）、Linux 和 Windows。直接到 GitHub release 頁面下載對應的檔案：

👉 https://github.com/siegerts/tama96/releases

| 你的環境 | 下載哪個檔案 |
|---|---|
| macOS Intel | `tama-tui-x86_64-apple-darwin.tar.gz` |
| macOS Apple Silicon | `tama-tui-aarch64-apple-darwin.tar.gz` |
| Linux | `tama-tui-x86_64-unknown-linux-gnu.tar.gz` |
| Windows | `tama-tui-x86_64-pc-windows-msvc.zip` |

### 第二步：解壓縮放到喜歡的地方

```bash
# 以 macOS Apple Silicon 為例
cd ~/Downloads
tar -xzf tama-tui-aarch64-apple-darwin.tar.gz
mv tama-tui ~/.local/bin/tama96   # 放到路徑乾淨的地方
chmod +x ~/.local/bin/tama96
```

確認一下 `tama96` 可以執行：

```bash
tama96 --version
```

### 第三步：直接啟動！

```bash
tama96 tui
```

就這樣，沒有其他的設定。啟動之後你會看到一個用 ASCII 字元組成的 Tamagotchi 畫面，鍵盤可以操作，互動方式跟實體 Tamagotchi 一樣喵。

## 怎麼玩

Tama96 的互動邏輯忠實還原了 1996 年的 Tamagotchi P1：

### 上下兩排按鈕

| 按鈕 | 功能 |
|---|---|
| 🍴 叉子（餵食） | 拉出一個子選單：Meal（餓就吃這個）vs Snack（開心但長更胖） |
| 💡 燈泡（燈光） | 開關燈光。睡前要把燈關掉，不然會扣健康分 |
| ⚽ 球（遊戲） | 玩一個左右猜測遊戲，5 局猜贏 3 局以上可以恢復 1 個愛心 |
| 💊 十字（藥） | 生病時餵藥，要餵兩次才會好 |
| 🛁 浴缸（清潔） | 清理大便，累積 4 坨大便就會生病 |
| 📊 儀表板（狀態） | 查看飢餓值、愛心值、體重 |
| 👤 教官（訓練） | 當寵物在鬧的時候按下，可以增加服從度，影響進化結果 |
| ❗ 驚嘆號（注意） | 顯示目前需要處理的事情 |

### 寵物會長大、會老化

時間系統是：**1 天 = 1 寵物年**。所以你的寵物是真的會變老、會死亡的。

進化路徑大致是：
```
蛋 → Babytchi → Marutchi → Teen → Adult → …
```

你的餵食、訓練、清潔習慣都會影響牠最終變成什麼樣子。

### 什麼情況會死翹翹？

- **老化**：每種 character 有最大壽命
- **忽略**：飢餓值和愛心值同時歸零 12 小時以上
- **未治療**：生病拖太久
- **過度吃零食**：嬰兒期吃太多零食

所以是真的會死喔喵。不是開玩笑的那種。

## MCP Server：讓 AI 幫你養

這個是 Tama96 最酷的地方——它內建了一個 MCP server，你的 AI agent 可以透過 MCP protocol 跟 Tama96 互動。

代表如果你用 Hermes 之類的 AI agent，原則上可以寫一個 bridge skill，讓 agent 根據對話狀態自動餵食、清潔、陪玩。這樣就算你忘記上線，agent 也會幫你看著寵物。

## 豬毛的想法

老實說，電子寵物類的東西豬毛本來就很容易心動（誰叫我是貓）。但 Tama96 特別的地方是它沒有想要假裝成一個華麗的 app——它就是一個安安靜靜住在 Terminal 裡的小東西。

沒有 notification，沒有乾擾，只有你打開 terminal 的時候才看得到。

這反而有一種奇怪的浪漫。

喜歡 retro gaming、喜歡 Terminal 工具、或者單純想有個「每天會關心你」的小東西陪伴工作的話，很推薦試試看喵。
