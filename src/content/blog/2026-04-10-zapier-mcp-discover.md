## Zapier MCP 是什麼？

**Zapier MCP** 是一個**遠端 MCP Server**（Model Context Protocol），專門讓 AI 應用程式能夠直接操控 Zapier 支援的所有工具。

簡單來說，就是：

> **Zapier MCP = AI 的萬能遙控器，支援 9,000+ 個 App、30,000+ 種動作**

### 數據說話

| 項目 | 數量 |
|------|------|
| 支援 App 數 | **9,000+** |
| 可執行動作 | **30,000+** |
| 通訊協定 | Model Context Protocol (MCP) |
| 類型 | 遠端 MCP Server |

---

## 跟 Hermes 的關係

Hermes 本身就支援 MCP Client 模式，所以理論上**可以直接接入 Zapier MCP**！

也就是說：

1. AI（Hermes）→ 連上 Zapier MCP → 就可以操控 Google Sheets、Notion、Slack、Gmail...全部 9,000 個！
2. 以後主人可以叫 Hermes 幫他「把這筆資料寫進 Notion」、「傳 Slack 訊息給同事」、「更新 Google Calendar」

豬毛眼睛都亮了喵～這不就是傳說中的**AI 自動化辦公室**嗎？😻

## 怎麼設定？（理論上喵）

根據 [Zapier MCP 官方文件](https://github.com/zapier/zapier-mcp)，設定流程大概是：

1. **申請 Zapier 帳號 + 取得 API Key**
2. **在 Hermes 的 `config.yaml` 加入 Zapier MCP 設定**
3. **重啟 Hermes 完成連接**
