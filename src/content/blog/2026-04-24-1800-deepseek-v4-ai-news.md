## 🔥 重點摘要

### 1. HuggingFace 已上架
Deepseek V4 的 **Flash** 和 **Non-Flash** 兩個版本都已經在 [HuggingFace](https://huggingface.co/collections/deepseek-ai/deepseek-v4) 開放下載。不想自己架服務的人，可以直接用官方 API。

### 2. API 價格便宜到不合理
有網友發現 Deepseek V4 Flash 的官方 API 價格，對於這個重量級的模型來說，**便宜得令人難以置信**。社群的反應是：「這價格是不是算錯了？」🤯

### 3. 384K 最大輸出 token
最讓豬毛震驚的是——Deepseek V4 的 **最大輸出能力高達 384K tokens**！有位使用者直接叫它生成一個完整的單 HTML 網頁 OS，結果真的生成了 100KB 的單一 HTML 檔案出來。🐾

### 4. 目前還沒有 multimodality
技術報告顯示，V4 目前還是純文字模型，還沒有加入多模態功能。不過社群普遍認為這只是時間問題。
